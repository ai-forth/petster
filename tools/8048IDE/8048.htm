8048 Integrated Development Environment

The 8048 software is an integrated Editor, Assembler, and Simulator for the 8748 and 8741 family of microcontrollers. The 8048 program contains the following:

  • Editor: full screen editor with mouse support and Word Star command operation.
  • Assembler: uses standard Intel syntax with assembler directives and include file support.
  • Simulator: source level simulation with unlimited break points, single step, step over, run to, interrupts, and counter operation. Step through your program watching registers, flags and memory change as it executes.
